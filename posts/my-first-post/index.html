Setting Up Python Environment

Author: David Feng
Sep 20, 2024

Install Python

Step 1: Download & Install Python

Visit the official Python website and download the installer for your operating system.

Step 2: Install Python Extension for Visual Studio Code

To use Python in VSCode, you need to install the Python extension. Please follow the official website for installation instructions.

Environment Setup in VSCode - Mac Version

Assuming VSCode is already installed, the process for setting up the Python environment is as follows:

  • Confirm the virtual environment path
  • Create a virtual environment
  • Install ipykernel
  • Add the virtual environment as a Jupyter kernel
  • Confirm the kernel has been successfully added
  • Select the kernel in Jupyter Notebook

Detailed explanation

1. Confirm the virtual environment path

If your virtual environment is named env_bio and is located in the project directory, the path to the virtual environment might be :

# Input
pwd
# Output
/path/to/your/project/env_bio

2. Create a virtual environment

Run the following command to activate your virtual environment :

# Input
python3 -m venv /path/to/your/project/env_bio
source /path/to/your/project/env_bio/bin/activate

When the virtual environment is successfully activated, you will see its name in the terminal prompt, for example :

# bash
(env_bio) yourusername@your-computer:~/path/to/project$

3. Install ipykernel

Installing ipykernel is done to add the Python virtual environment to Jupyter Notebook, allowing it to be used as a kernel by Jupyter.

# Input
pip3 install ipykernel

4. Add the virtual environment as a Jupyter kernel

# Input
python3 -m ipykernel install --user --name env_bio --display-name "Python 3.12 (env_bio)"
  • --name : This is the internal kernel name (e.g., env_bio), used by Jupyter for identification.it.
  • --display-name : The name shown in the Jupyter Notebook kernel menu (e.g., ‘Python 3.12 (env_bio)’), visible to users.

5. Confirm the kernel has been successfully added

To check if the virtual environment has been successfully added to Jupyter’s kernel list, run the following command :

# Input
jupyter kernelspec list
# Output
Available kernels:
    python3         /path/to/some/kernel
    env_bio         /Users/yourusername/path/to/your/project/env_bio

6. Select the kernel in Jupyter Notebook

  • Open VSCode and open a Jupyter Notebook file (.ipynb).
  • Click the Kernel dropdown at the top of the Notebook page.
  • Select the newly added ‘Python 3.12 (env_bio)’ kernel from the dropdown menu.
