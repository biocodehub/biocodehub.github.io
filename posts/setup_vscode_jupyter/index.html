Install Visual Studio Code & Set Up Jupyter Notebook in VSCode

Step 1: Install Visual Studio Code

Go to the official VSCode website and download the installer for your operating system.

Step 2: Install the Jupyter and Python Extensions

1. Install the Jupyter Extension :

  • Click on the Extensions icon on the left sidebar in VSCode (or press Cmd+Shift+X).
  • Search for “Jupyter” in the Extensions Marketplace.
  • Click Install next to the “Jupyter” extension.

Step 3: Open or Create a Jupyter Notebook in VSCode

1. Open the Command Palette :

Press Cmd+Shift+P (on macOS) or Ctrl+Shift+P (on Windows/Linux) to open the Command Palette in VSCode.

2. Create a New Jupyter Notebook :

In the Command Palette, type Create : New Jupyter Notebook and select it from the list.
